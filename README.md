## 🏛️ Executive Value Proposition

Most financial models exist as fragile, manual spreadsheets prone to hardcoded errors, formula drift, and key-person risk. **Valuation Studio** bridges rigorous financial economics with production-grade software engineering.

It provides an automated, version-controlled Python projection engine that generates live, formula-driven Excel models. This architecture enables programmatic scenario testing, peer-group outlier trimming, and boardroom-ready reporting—translating complex technical methodologies directly into actionable intrinsic value insights and risk mitigation strategies.

```text
[ Raw Financials (API/CSV) ] ──> [ Valuation Engine (Python) ] ──> [ Synchronized Excel Model (.xlsx) ]
                                            │
                                            ├──> [ Unlevered DCF & WACC Sensitivities ]
                                            └──> [ Boardroom Executive Briefing (CLI/Rich) ]
```

## ✨ Core Capabilities

* **Dynamic Balance Sheet Equilibrium:** Implements an automated Cash and Revolver (credit facility) plug, ensuring $Assets = Liabilities + Equity$ across all projection horizons without manual intervention.
* **Algorithmic Unlevered DCF:** Calculates intrinsic equity value using mid-year discounting conventions for operational cash flows and precise end-of-period discounting for Gordon Growth Terminal Value.
* **Boardroom-Ready Excel Export:** Transforms numpy arrays into formatted, heavily stylized `.xlsx` workbooks with live formulas (`openpyxl`), allowing investment committees to audit assumptions natively in Excel.
* **Scenario Pipeline Management:** Instantly pivot between Bull, Base, and Bear cases to assess Margin Expansion, CLV impacts, and downside risks.
* **LaTeX Methodology Documentation:** Generates executive-grade mathematical methodology reports compiled automatically via `pdflatex`.

## 🚀 Quickstart

### 1. Installation

Clone the repository and install the package in development mode along with its dependencies:

```bash
git clone [https://github.com/thanhan25/equity-valuation-dcf-model.git](https://github.com/thanhan25/equity-valuation-dcf-model.git)
cd equity-valuation-dcf-model
pip install -e .[dev]
pre-commit install
```

### 2. Execute the Valuation Pipeline

Run the end-to-end engine for NVIDIA (`NVDA`) under the **Bull Case** scenario, outputting a fully linked Excel model:

```bash
valstudio run --ticker NVDA --scenario bull --output models/examples/sample_valuation_nvda.xlsx
```

*The CLI will automatically fetch financials, project the 3-statements over 5 years, run the DCF math, and export a formatted `sample_valuation_nvda.xlsx` ready for executive review.*

## 📐 Mathematical & Accounting Architecture

### Unlevered Free Cash Flow ($FCFF$)

The quantitative engine computes cash generated by operations after accounting for reinvestment in long-term assets and working capital via exact accounting identities:

$$
FCFF = EBIT \cdot (1 - t) + D\&A - \Delta NWC - CapEx
$$

### Weighted Average Cost of Capital ($WACC$)

Discount rates are dynamically computed to reflect target capital structures, capturing the true opportunity cost of capital:

$$
WACC = \frac{E}{V} \cdot R_e + \frac{D}{V} \cdot R_d \cdot (1 - t)
$$

### Two-Variable Sensitivity Matrix

Calculates a $5 \times 5$ intrinsic share price grid varying $WACC$ against the Terminal Growth Rate ($g$), ensuring rigorous risk mapping and margin-of-safety analysis.
